INTRODUÇÃO

O tema deste trabalho é o MoveGoal — uma plataforma web de análise de desempenho para jovens atletas amadores. O objetivo é demonstrar como uma aplicação simples e acessível pode ajudar jovens entre os 12 e os 18 anos a acompanhar o seu progresso desportivo, usando apenas tecnologia web estática.
